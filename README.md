# Webooster

「Web + Booster」なWebデザインの開発環境を構築します。

## Installation

以下のコマンドを実行します。

```
git clone https://github.com/koluku/webooster.git
npm install
npm install -D babel
```

その後、package.jsonの内容を書き換えてください。

## Usage

```
gulp
```

```
gulp release
```
